### Installation

1. Clone the repository
   ```sh
   git clone https://github.com/xtrahuman/Okpara-Tochukwu-react.git
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
3. Run server
   ```sh
   npm start
   ```
